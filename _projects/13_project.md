# Objetivo general
Diseñar un sistema de monitoreo de CO2 de bajo costo para informar a los ocupantes de los espacios cerrados el nivel de concentración de CO2 a fin de reducir la probabilidad de contagio de SARS-COV-2.

# Objetivos específicos

1. Establecer la comunicación entre un dispositivo de bajo costo CPU RISC de 32-bit y un sensor de CO2 NDIR.
2. Mostrar los niveles de concentración de CO2 de un espacio cerrado en una interfaz de visualización. 
3. Diseñar una interfaz de comunicación inalámbrica entre el dispositivo de bajo costo.
4. Implementar el sistema de monitoreo en un aula cerrada de la universidad.
